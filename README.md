## What It Is

A locally-sovereign multi-model debate engine implementing the 
FI-13 Specialist Debate pattern (P10 AVP) and trust-weighted 
consensus synthesis (P11 CSP). Three heterogeneous local LLMs 
debate any topic through a structured ASSERT → COUNTER → 
VALIDATE cycle, producing a binding verdict with provenance.

All inference runs locally via Ollama. No cloud dependencies. 
No data leaves the host machine.
